Crafted for Scroll Lovers

This gallery demonstrates a layered parallax effect powered by JavaScript with accessibility and performance in mind. Use the theme toggle to switch between light and dark modes. All images are public domain or free-to-use photography.

  • Responsive layout via Bootstrap grid
  • Utilities and theming via Tailwind
  • Smooth parallax using requestAnimationFrame
// Toggle theme
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
const stored = localStorage.getItem('theme');
const initial = stored ? stored === 'dark' : prefersDark;
document.documentElement.classList.toggle('dark', initial);
